Birds Evolution Simulation

This is a side project I developed to learn Rust & WebAssembly.

It implements a simple NeuralNetwork that simulate the brain of an animal looking for food. Animals brain evolve using a standard genetic algorithm that mix up brains Neural Network in each new population generation. You can tweak some parameters in this simulation and see how the evolution changes according to them.

Have fun!
